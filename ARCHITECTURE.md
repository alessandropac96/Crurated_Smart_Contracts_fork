## Component Interaction Flow

### Query Execution Flow

```
1. Client Request
   вҶ“
2. QueryExecutor receives QueryParams
   {
     filters: [FILTER_BY_OWNER],
     filterParams: [encode(ownerAddress)],
     sortField: TOKEN_ID,
     sortDirection: DESC,
     offset: 0,
     limit: 10,
     owner: ownerAddress
   }
   вҶ“
3. STEP 1: Get Filtered & Sorted IDs
   вҶ“
   3a. QueryExecutor вҶ’ _getAllTokenIds()
       Returns: [1, 2, 3, 4, 5]
   вҶ“
   3b. QueryExecutor вҶ’ FilterManager.applyFilters()
       Input: [1, 2, 3, 4, 5]
       Filters: [FILTER_BY_OWNER]
       Returns: [1, 2, 5] (tokens owned by address)
   вҶ“
   3c. QueryExecutor вҶ’ SortManager.sort()
       Input: [1, 2, 5]
       Sort: TOKEN_ID, DESC
       Returns: [5, 2, 1]
   вҶ“
   3d. Apply Pagination
       Input: [5, 2, 1]
       Offset: 0, Limit: 10
       Returns: [5, 2, 1] (all fit in page)
   вҶ“
4. STEP 2: Fetch Data with Caching
   вҶ“
   4a. For each tokenId in [5, 2, 1]:
       вҶ“
       DataProvider.getTokenData(tokenId, owner)
       вҶ“
       Check Cache:
       в”ңв”Җ If cached & valid вҶ’ Return cached data (CACHE HIT)
       в””в”Җ If not cached вҶ’ Fetch from Crurated (CACHE MISS)
          вҶ“
          Fetch:
          - tokenId
          - cid (from Crurated.cidOf())
          - balance (from Crurated.balanceOf())
          - latestStatus (from tracking)
          - statusCount (from tracking)
          вҶ“
          Store in cache
          вҶ“
          Return TokenData
   вҶ“
5. Build QueryResult
   {
     tokenData: [TokenData5, TokenData2, TokenData1],
     totalCount: 3,
     hasMore: false
   }
   вҶ“
6. Return to Client
```

## Summary

This architecture provides:

вң… **Clean Separation**: Each component has a single responsibility
вң… **Flexibility**: Components can be upgraded independently
вң… **Efficiency**: Two-step pattern minimizes gas usage
вң… **Scalability**: Caching reduces repeated data fetching costs
вң… **Extensibility**: Easy to add new filters, sort fields, or data sources
вң… **Testability**: Each component can be tested in isolation
